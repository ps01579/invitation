# 산수연 초대장 정적 웹사이트

웹서버에 바로 업로드해서 사용할 수 있는 정적 사이트입니다.

## 구성 파일
- `index.html`: 메인 페이지
- `styles.css`: 디자인 스타일
- `script.js`: 버튼, 복사 기능 등 동작
- `site-data.js`: 행사 정보, 문구, 계좌, 지도 링크를 수정하는 설정 파일

## 바로 배포하는 방법
### 1) 일반 웹호스팅 / 사내 웹서버
압축을 해제한 뒤 파일 전체를 웹 루트에 그대로 업로드하면 됩니다.
예시:
- Nginx: `/usr/share/nginx/html/`
- Apache: `/var/www/html/`

### 2) GitHub Pages
- 새 저장소 생성
- 압축을 해제한 파일을 업로드
- `Settings > Pages` 에서 배포 브랜치 선택
- 몇 분 후 공개 URL 생성

## 꼭 수정해야 하는 부분
`site-data.js` 파일만 수정하면 대부분 원하는 형태로 바꿀 수 있습니다.

### 수정 항목
- 주인공 성함
- 행사 일시
- 장소명 / 주소
- 초대 문구
- 연락처
- 계좌번호
- 네이버지도 / 카카오맵 링크
- 갤러리 이미지

## 사진 바꾸는 방법
현재는 외부 이미지 URL이 들어 있습니다.
직접 사진을 쓰려면:
1. `assets/photos/` 폴더를 만들고
2. 사진 파일을 넣은 뒤
3. `site-data.js` 의 `gallery.src` 값을 예를 들어 `./assets/photos/family1.jpg` 처럼 바꾸면 됩니다.

## 추천 추가 작업
- `assets/og-image.png` 추가: 카카오톡 공유 썸네일용
- 실사용 전 문구와 연락처 최종 점검
- 실사용 주소에 맞는 지도 링크 교체

## 주의
이 사이트는 정적 사이트라서 RSVP 저장 같은 서버 기능은 없습니다.
필요하면 추후 Google Form 연동 또는 간단한 서버 API를 붙일 수 있습니다.
